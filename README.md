# 🌼 Portfolio

> **작은 씨앗에서 시작해 꾸준한 성장으로 꽃을 피우는 프론트엔드 개발자, 이예원입니다.**

React를 기반으로 제작한 개인 포트폴리오 웹사이트입니다.
프로젝트 경험과 기술 스택을 한눈에 확인할 수 있도록 구성했으며,
직접 구현한 프로젝트와 성장 과정을 담았습니다.

# 📂 Folder Structure

```text
portfolio-react
├── public                    # 정적 파일 및 이미지
│   ├── image                 # 프로젝트 이미지 및 리소스
│   ├── favicon.png           # 파비콘
│   ├── index.html            # HTML 템플릿
│   ├── logo192.png           # PWA 아이콘 (192px)
│   ├── logo512.png           # PWA 아이콘 (512px)
│   ├── manifest.json         # PWA 설정 파일
│   └── robots.txt            # 검색 엔진 크롤링 설정
│
├── src
│   ├── components            # 재사용 가능한 UI 컴포넌트
│   │   ├── AboutValues.jsx
│   │   ├── Contact.jsx
│   │   ├── Header.jsx
│   │   ├── Hero.jsx
│   │   ├── ProjectDetail.jsx
│   │   ├── Projects.jsx
│   │   └── Skills.jsx
│   │
│   ├── data                  # 프로젝트 및 기술 데이터(JSON)
│   │   ├── projects.json
│   │   ├── skills.json
│   │   └── values.json
│   │
│   ├── pages                 # 페이지 컴포넌트
│   │
│   ├── styles                # SCSS 스타일 파일
│   │   ├── About.scss
│   │   ├── common.scss
│   │   ├── Contact.scss
│   │   ├── Header.scss
│   │   ├── Hero.scss
│   │   ├── ProjectDetail.scss
│   │   ├── Projects.scss
│   │   └── reset.scss
│   │
│   ├── App.js                # 애플리케이션 루트 컴포넌트
│   ├── App.css               # App 스타일
│   ├── index.js              # React 진입점
│   ├── index.css             # 전역 스타일
│   └── logo.svg              # 기본 로고
│
├── .gitignore                # Git 제외 파일 설정
├── package.json              # 프로젝트 및 의존성 정보
├── package-lock.json         # 의존성 버전 관리
└── README.md                 # 프로젝트 소개 문서
```
